# Product Overview

## The Referee - Cloud Service Comparator

A decision-support tool that helps developers and businesses make informed decisions about cloud infrastructure choices by providing real-time pricing data, balanced pros/cons analysis, and clear trade-offs between different cloud services.

### Core Purpose
- Compare cloud services across AWS, Google Cloud, and Azure
- Provide live pricing data (currently AWS via public API)
- Offer balanced analysis with pros, cons, and trade-offs
- Help users make informed cloud architecture decisions

### Current Features
- **Serverless Compute Comparison**: AWS Lambda vs Google Cloud Functions vs Azure Functions
- **AWS Storage Comparison**: S3 vs EFS vs EBS
- **Multi-region pricing**: US East, Asia Pacific (Mumbai), EU (Ireland)
- **Live pricing integration**: Real-time AWS pricing via public API
- **Fallback pricing**: Graceful degradation when APIs are unavailable

### Target Users
- Developers evaluating cloud options
- Solution architects planning infrastructure
- Business analysts comparing costs
- CTOs making strategic cloud decisions

### Key Value Proposition
Provides unbiased, data-driven cloud service comparisons with real-time pricing and practical trade-off analysis to accelerate cloud decision-making.
